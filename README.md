# Qodalis CLI — Vanilla Example

## Quick Start

```bash
npm install
npm run dev
```

## What's Included

- `@qodalis/cli` — Framework-agnostic CLI engine (CliEngine)
- `@qodalis/cli-guid` — GUID generator plugin
- `@qodalis/cli-files` — File browser plugin
- `@qodalis/cli-users` — User management plugin
